# SensorPanelToo

自定义硬件监控面板，可自由设计仪表盘界面并全屏渲染到任意显示器。

## 功能

- **拖拽式主题编辑器** — 画布上自由添加/排列环形仪表、进度条、数字显示、折线图、传感器标签
- **实时硬件监控** — CPU/GPU/内存/存储等传感器数据，500ms 刷新
- **全屏渲染** — 支持多显示器，自动适配比例（letterbox/pillarbox）
- **背景图支持** — 自定义背景色 + 背景图片，缩放/偏移/居中/适应
- **组件属性编辑** — 颜色（HSV 色盘）、缩放、字体、传感器绑定等
- **主题持久化** — JSON 保存/加载，文件路径对用户透明

## 技术

.NET 8 + WPF · LibreHardwareMonitorLib · HSV 取色器 · Win32 多显示器定位

## 用法

1. 启动 → 主控制台
2. 「主题」页 → 选择主题 → 「渲染」（双击快速渲染）
3. 「编辑」→ 打开主题编辑器，修改布局和属性
4. 「新建」→ 创建空白主题
5. 「调试」页 → 单组件实时预览调试
